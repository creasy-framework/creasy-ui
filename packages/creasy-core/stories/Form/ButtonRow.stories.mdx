## Button Row
> Responsive button layout, usually used in footer section of dialog or form. Resize browser window to small size to see how the buttons render on mobile devices.

```tsx live expanded
<ButtonRow
  primary={[
    <Button key="save-btn" color="secondary" size="m">Save</Button>,
    <Button key="delete-btn" color="danger" size="m">Delete</Button>
  ]}
  secondary={[<Button key="cancel-btn"  size="m">Cancel</Button>]}
>
</ButtonRow>
```
